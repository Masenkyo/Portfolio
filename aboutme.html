About Me

Hallo! Ik ben Sil Wever, ik ben geboren op zeven juli 2005 en ik woon in Den Helder. Ik woon met mijn moeder, vader, zus, kat en hond. Ik zit nu op Mediacollege in Amsterdam. Mijn hobbies zijn gamen, drummen, atletiek en games developen.

Ik hoop een stage bedrijf te vinden dat binnen 2 uur reistijd te bereiken is en ik vind best veel soorten games leuk, dus het word vast niet moeilijk om mijn interesse te laten zien bij meerdere subjects. Ik gebruik vooral C# in Unity en ik ben best goed met C#. Ook heb ik in Unity aan Mirror Networking gewerkt op 2 verschillende games. Wat ik nu het graagste zou willen leren is C++.

Ik kan erg goed met anderen samenwerken, maar ook erg goed alleen, want dan kan ik erg goed focussen op wat ik mee bezig ben. Als ik begin met code maken, probeer ik eerst een werkende code te maken en vooruit te denken over hoe compatible ik het kan maken met latere aanpassingen, dan netter maken zodat anderen het makkelijker kunnen begrijpen. Het moet ook erg makkelijk zijn om dingen erin aan te passen. Graag leer ik nieuwe dingen te gebruiken zoals nieuwe talen (C++, Java etc.) en anderen engines (Unreal Engine, Godot etc.).

Graag zou ik bij een bedrijf stage lopen/werken die similar interesses hebben, maar nog belangrijker is een leuke sfeer, vrolijke mensen met net zoveel passie als ik heb, en als laatste dat ik welkom voel.
