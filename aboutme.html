About Me

Hallo! Ik ben Sil Wever, ik ben geboren op zeven juli 2005 en ik woon in Den Helder. Ik woon samen met mijn moeder, vader, zus, kat en hond. Ik zit nu op school bij Mediacollege in Amsterdam. Mijn hobby's zijn gamen, drummen, atletiek en games maken.

Als Unity Gameplay Developer maak ik dingen zoals het multiplayer maken in games en dan alles synchroniseren. Een voorbeeld van een game die ik multiplayer heb gemaakt is Operation Starfall. Ook vind ik het leuk om te werken aan mechanics zoals in elk van mijn tot nu toe gemaakte games. Ik vind het leuk om de mechanics goed te laten voelen voor de speler en dat ik alles vloeiend kan laten lopen voor een goede speelervaring. Zoals je ziet, heb ik vooral 2D games gemaakt, want ik vind dat leuk, maar ik kan ook 3D games maken. Ik weet wat de verschillen zijn tussen 2D en 3D games maken, ik ben even goed in beide. Als ik bij een stage bedrijf werk, wil ik graag aan de mechanics werken en/of aan de multiplayer.

Ik kan erg goed met anderen samenwerken, maar ook erg goed alleen, want dan kan ik erg goed focussen op wat ik mee bezig ben. Als ik begin met code maken, probeer ik eerst een werkende code te maken en vooruit te denken over hoe compatibel ik het kan maken met latere aanpassingen, dan netter maken zodat anderen het makkelijker kunnen begrijpen. Het moet ook erg makkelijk zijn om dingen erin aan te passen. Ik wil leren hoe ik in andere engines zoals unreal engine en godot kan werken en ook andere code talen te leren zoals c++.

Ik wil bij een bedrijf stage lopen/werken die vergelijkbare interesses hebben, maar het is nog belangrijker dat de sfeer goed is. Ook is het belangrijk dat ik met mensen werk die net zoveel passie hebben voor game development als ik.
