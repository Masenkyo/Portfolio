About Me

Hallo! Ik ben Sil Wever, ik ben geboren op 2005-7-7 en ik woon in Den Helder. Ik woon met mijn moeder, vader, zus, kat en hond. Ook had ik 3 vogels, 2 konijnen en mijn beste vriend Billy (hond), helaas zijn ze niet meer bij ons.

Sinds ik jong was vond ik games altijd erg leuk om te doen, maar ik deed ook aan atletiek. Ik vond het gelukkig allebij best leuk om te doen, ik speelde toen ik nog op de basisschool zat veel roblox, minecraft, gta v, call of duty etc.

Toen ik op de middelbare school zat, speelde ik steets meer op de pc dan op de playstation 4. Vanaf toen begon ik na te denken over hoe games nou eigenlijk gemaakt werden. Ik vond erg veel dingen over program languages en game engines, dat was wel een beetje ingewikkeld voor mij op de middelbare school en ik vond gamen leuker dan uitzoeken hoe ik zelf een game kon maken, maar sinds ik in de 4de zat had ik er steeds meer intresse in.

Toen kwam ik terecht op MediaCollege op de Software Development opleiding, zodat ik meer kennis over games maken kreeg en misschien mijn eigen games kon maken, want ik had allemaal ideeen voor games die ik dan zou willen maken en had er erg veel zin in. Ik heb al erg veel geleerd over code languages en hoe ik dat kan gebruiken om games mee te maken en wiskunde erin te verwerken. Ik heb c# heb beste onder de knie, maar ik kan ook een beetje c++, python en misschien java (misschien omdat ik heel lang geen java meer heb gedaan).

Ik zou later wel een Indie Gamedeveloper willen worden en gewoon mijn eigen ideeen tot leven werken, maar misschien is het wel een goed idee om eerst met een bedrijf te werken aan games om werk ervaring mee te nemen wat wel handig kan zijn, dan leer ik nog ook van anderen. Het originele idee was om helemaal alleen games te maken, maar ik en mijn vrienden vonden het misschien een leuk idee om met z'n 5en (4 game devs, 1 game artist) een eigen Indie Game Bedrijf te maken en samen coole games gaan maken als we klaar zijn met onze opleidingen.
